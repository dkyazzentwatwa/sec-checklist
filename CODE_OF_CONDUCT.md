# Code of Conduct

Rights Shield is built by and for community members. We expect all contributors to participate respectfully and create a welcoming environment.

## Expected Behavior
- Be respectful and constructive.
- Assume good intent and ask clarifying questions.
- Focus on impact, especially for affected communities.

## Unacceptable Behavior
- Harassment, discrimination, or hate speech.
- Doxxing or sharing private information.
- Disruptive behavior that derails collaboration.

## Reporting
If you experience or witness harmful behavior, contact the project maintainers through the repository communication channels. Reports will be handled confidentially.
