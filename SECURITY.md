# Security Policy

## Reporting a Vulnerability

Please report security issues via GitHub Security Advisories:
https://github.com/dkyazzentwatwa/right-guard/security/advisories/new

If you are unable to use GitHub advisories, open a private issue with minimal details and we will follow up.

## Supported Versions

We currently support the latest `main` branch and the most recent release.

## Disclosure

We ask that you give us a reasonable amount of time to investigate and address security issues before public disclosure.
